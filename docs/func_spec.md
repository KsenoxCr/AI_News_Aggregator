# Functional Specification

## Auth

- Email only auth, storage


## Usage flows

Landing:
    -> Welcome Screen
        -> input email addr for login/signup

Login/Signup:
    -> Input email addr
    -> emphemeral login link send to email inbox

After signup: (Same functions found in settings)
    -> News Source selection
        -> Select from list
        -> Add new source
    -> AI Model selection: endpoint URL + API key (BYOK)
    -> Interest specification
        -> Choose from list or text freeform

News Headers List
    -> Show timeframes articles
    -> Change timeframe
    -> Open news arcticle to view

Article modal
    -> Read AI summary
    -> Open original article (publisher's website)
